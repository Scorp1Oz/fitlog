# fitlog 🏋️

Особистий **офлайн-трекер силових тренувань** для Android. Усі дані живуть на
телефоні — без сервера й щомісячної плати за хостинг. Водночас це
портфоліо-проєкт: тоннаж, оцінка 1ПМ, теплова карта тіла, експорт CSV.

> Стиль: темний + лайм. Brutalist-естетика, великі цифри, пласкі поверхні.

---

## Технологічний стек

| Шар | Інструмент |
|---|---|
| Платформа | **Expo (React Native)**, SDK 56 · TypeScript |
| Стилі | **NativeWind v5** + Tailwind v4 (токени в `global.css`) |
| Шрифти | Bebas Neue · IBM Plex Sans · IBM Plex Mono |
| База даних | **expo-sqlite** — єдине джерело правди, повний офлайн |
| Міграції | власний раннер на `PRAGMA user_version` (схема v6) |
| Профілі | мультипрофільність: кожен акаунт має власні дані (`profile_id`) |
| Навігація | **expo-router** + **react-native-pager-view** (свайп між вкладками) |
| Рантайм-стан | **Zustand** (активна сесія тренування, активна пробіжка) |
| Захист входу | **expo-local-authentication** + **expo-secure-store** |
| Картинки | **expo-image** (прев'ю вправ, аватари, фото вправ) |
| Кардіо / GPS | **expo-location** + **expo-task-manager** (запис треку у фоні) |
| Карти | **MapLibre** + векторні тайли OpenFreeMap (без API-ключа) |
| Сповіщення | **expo-notifications** (фоновий сигнал таймера відпочинку) |

## Поточний стан

**Фаза 0 — каркас (завершено):**

- [x] NativeWind v5 + дизайн-токени (кольори, шрифти)
- [x] Екран-замок: біометрія + прапорець у SecureStore (з обробкою браузера)
- [x] Модуль БД + міграційний раннер (схема v2: profiles, exercises, routines,
  sessions, session_sets; `avatar_uri`)

**Фаза 1 — ядро трекера (в основному готово):**

- [x] Реєстрація / логін на таблиці `profiles` (хешований пароль, активний
  профіль, «залишитись у системі», запит біометрії після реєстрації)
- [x] Стилізований екран входу (радіальний градієнт, матове скло, анімовані
  частинки на reanimated)
- [x] Нижні вкладки (Головна · Тренування · Аналітика · Біг) — горизонтальний
  пейджер зі свайпом, плаваючий бар; фірмові лаймові градієнти фону
- [x] Профіль: постійна шапка, модалка-перегляд, редагування (ім'я, аватар із
  видаленням, дата народження + автоматичний вік)
- [x] Бібліотека вправ: 873 вправи (free-exercise-db), українська таксономія,
  пошук/фільтр; **створення власних вправ** (фото + категорії); **переклад
  назв вправ** (гібрид: словник + токенний рушій; англ. `name` збережено)
- [x] **Логування тренування**: вільна сесія, підходи (вага × повтори),
  таймер відпочинку, збереження в `sessions` / `session_sets` (Zustand)
- [x] **Історія**: календар місяця (дні з тренуваннями підсвічені), деталі
  дня та сесії (вправи, підходи, тоннаж)
- [x] **Прогресія**: оцінка 1ПМ (Еплі), персональні рекорди, графік прогресу

**Фаза 3 — рутини та програми (готово):**

- [x] **Рутини** (шаблони): набір вправ із цілями (підходи × діапазон повторів);
  «почати за рутиною» → ведений runner (одна вправа на екран, прогрес, таймер)
- [x] **Програми**: розклад рутин по днях тижня, активна програма на профіль;
  екран «Тренування» показує тижневий розклад і запускає день
- [x] **Показові плани**: дві демо-програми засіваються однією кнопкою (кнопка
  ховається після першого використання; відновлюється в налаштуваннях)
- [x] **Головний екран**: бокс «Сьогоднішнє тренування» — рутина дня активної
  програми із запуском у тап

**Фаза 4 — кардіо / біг (готово):**

- [x] **GPS-трекінг пробіжки**: запис треку у фоні (foreground service) —
  дистанція рахується навіть зі згорнутим застосунком
- [x] **Карта** на MapLibre (темний стиль, тайли OpenFreeMap без ключа): лінія
  маршруту, точки старту/фінішу; дозвіл локації запитується на вході
- [x] **Підсумок пробіжки** з лаймовим переходом-завісою (дистанція/час/темп +
  карта), пробіжки лягають у той самий календар історії
- [x] **Фоновий таймер відпочинку**: локальне сповіщення зі звуком/вібрацією,
  коли застосунок згорнутий (JS-таймер у фоні зупиняється)

- [ ] Графіки об'єму/тоннажу · секція «Тіло» (вага, заміри, фото) · експорт CSV

Повна дорожня карта: [`docs/fitlog_plan.md`](docs/fitlog_plan.md).

## Запуск

```bash
npm install
npx expo start        # далі: натисни 'a' (Android), 'w' (веб) або скануй QR в Expo Go
```

Потрібен лише Node.js і застосунок **Expo Go** на телефоні (Android, той самий
Wi-Fi). Android Studio не обов'язковий.

> ⚠️ Біометрія працює на пристрої з налаштованим відбитком/обличчям. У браузері
> екран-замок не блокує (передбачено).

## Структура

```
src/
  app/          # маршрути expo-router: (tabs)/ пейджер-хост, екрани
                #   workout-session, routine-run, exercises, exercise-*,
                #   routines/routine-*, programs/program-*, day, session,
                #   profile, settings
  screens/      # екрани-сторінки пейджера (Home/Workout/Analytics/Run)
  auth/         # акаунти: логін/реєстрація, Context, екран входу + декор
  lock/         # екран-замок: логіка, Context, UI, «ворота»
  db/           # SQLite: схема, раннер, profiles, exercises, sessions,
                #   routines, programs, seed-demo, provider
  workout/      # активна сесія: Zustand-стор (free|routine), таймер відпочинку,
                #   хук фонового сповіщення таймера
  run/          # кардіо: Zustand-стор пробіжки, фоновий таск GPS, контроль
                #   старт/стоп, карта MapLibre, bootstrap локації
  exercises/    # переклад таксономії + назв вправ, спільний пікер
  profile/      # аватар, модалка профілю (з активною програмою), хелпери
  theme/        # централізована палітра кольорів (джерело для тем)
  components/   # спільні компоненти (таб-бар, ScreenTitle, календар, шапки,
                #   фон, лаймова завіса, свічіння кнопок)
  lib/          # утиліти (формати дат/вік, дні тижня, сила/1ПМ, гео/темп,
                #   сповіщення, адаптивні розміри)
global.css      # токени дизайн-системи (@theme) + база Tailwind
assets/         # exercises.json (база вправ) + іконки
docs/           # план, дизайн-система, навчальні конспекти сесій
```

## Збірка APK

Проєкт має теку `android/` (prebuild), тож офлайн-APK збирається локально через
Gradle — без Metro і без встановлення на пристрій:

```powershell
cd android
.\gradlew assembleRelease
```

Готовий файл: `android\app\build\outputs\apk\release\app-release.apk` —
універсальний APK із вбудованим JS-бандлом. Кидаєш у хмару → качаєш на телефон →
ставиш (увімкнувши «встановлення з невідомих джерел»).

> Release підписується дебаг-ключем шаблону Expo — годиться для особистого
> тесту/портфоліо, але не для Play Store. Альтернатива в хмарі: `eas build -p
> android --profile preview` (buildType: apk).
