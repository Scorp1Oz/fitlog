# fitlog — Дизайн-система та макети
Стиль: темний + лайм (перенесено з FitGuide / training-теми). Для відтворення у Figma.

---

## 1. Кольори (токени)
Скопіюй як Color Styles у Figma.

| Токен | HEX | Призначення |
|---|---|---|
| `bg` | `#0D0D0D` | основний фон екрана |
| `surface` | `#161616` | картки, панелі, поля вводу |
| `surface-2` | `#1E1E1E` | вкладені блоки, header |
| `border` | `#222222` | рамки, роздільники |
| `faint` | `#2A2A2A` | неактивні номери, дуже тьмяний текст |
| `lime` | `#C8F135` | акцент, активний стан, головні цифри, CTA |
| `orange` | `#FF4D1C` | вторинний акцент (фокус дня, попередження) |
| `text` | `#E8E8E8` | основний текст |
| `text-muted` | `#888888` | підписи, описи |
| `text-dim` | `#555555` | мітки, мета-інфо |
| `success` | `#7BC043` | виконаний підхід |
| `on-lime` | `#0D0D0D` | текст на лаймовій кнопці |

Акцентні фони: лайм 3% (`rgba(200,241,53,0.03)`) для підказок, лайм 8% для активних рядків.

## 2. Типографіка
Усі шрифти безкоштовні (Google Fonts → у Figma і через `@expo-google-fonts` у коді — повний збіг).

- **Bebas Neue** — дисплейні заголовки, великі цифри (вага, рекорди, статистика).
- **IBM Plex Sans** — основний текст, назви вправ, кнопки.
- **IBM Plex Mono** — мітки, eyebrow-підписи, час/таймер, табличні цифри.

**Тайп-скейл (мобільний):**
| Стиль | Шрифт | Розмір / інтерліньяж | Трекінг |
|---|---|---|---|
| Display | Bebas Neue | 40 / 40 | +1 |
| H1 (екран) | Bebas Neue | 32 / 34 | +1 |
| H2 (секція) | Bebas Neue | 24 / 26 | +1 |
| Stat-number | Bebas Neue | 28 / 28 | 0 |
| Body | IBM Plex Sans 400 | 15 / 22 | 0 |
| Body-strong | IBM Plex Sans 600 | 15 / 22 | 0 |
| Caption | IBM Plex Sans 400 | 12 / 16 | 0 |
| Label/Eyebrow | IBM Plex Mono 400 | 10 / 12 | +3, UPPERCASE |
| Mono-meta | IBM Plex Mono 400 | 11 / 14 | +1 |

## 3. Сітка та форма
- **Артборд:** 360 × 800 (Android) або 390 × 844 — обери один і тримайся.
- **Базовий відступ:** 8pt сітка (4, 8, 12, 16, 24, 32).
- **Поля екрана:** 16 px ліворуч/праворуч.
- **Радіуси:** 0–4 px (стиль майже прямокутний, як у FitGuide). Картки — radius 2.
- **Рамки:** 1 px `border`. Акцент картки — лайм смужка 3 px ліворуч (як `.exc::before`).
- **Тіней немає** — глибина через колір поверхні й рамки (flat-стиль).

## 4. Компоненти (специфікація)

**Bottom Tab Bar** — фон `surface`, верхня рамка `border`. Іконка + мono-підпис. Активна — `lime`, неактивна — `text-dim`.

**Screen Header** — eyebrow (Label, `text-dim`) над H1 (Bebas, `text`). Праворуч може стояти ряд Stat-tiles.

**Stat Tile** — Bebas-число (`lime`) + Mono-підпис під ним (`text-dim`). Для статистики тренування/тіла.

**Chip / Pill** — Mono 10, UPPERCASE, рамка 1px кольору категорії, прозорий фон. Для фільтрів і тегів м'язів.

**Exercise Card (бібліотека)** — рядок: прев'ю (expo-image, 48×48, radius 2) · назва (Body-strong) + м'яз/обладнання (Caption, `text-dim`). Hover/press — рамка світлішає.

**Set Row (екран логування)** — головний компонент:
```
[#]   prev: 60×8    [ вага ]  ×  [ повт ]   [✓]
```
№ підходу (`faint`), «минулого разу» (Mono, `text-dim`), два поля вводу (Bebas-число), кнопка-галочка. Виконаний рядок — ліва смужка `success`.

**Primary Button** — фон `lime`, текст `on-lime` (Body-strong). Press — лайм трохи темніший.
**Secondary Button** — прозорий, рамка `border`, текст `text`.

**Input Field** — фон `surface`, нижня рамка 1px `border`, фокус — рамка `lime`. Числові — вирівняні по центру, Bebas.

**In-screen Tabs** (горизонтальні, як `.wnav`) — Mono-мітки, активна з нижньою лаймовою рискою 2px, скрол по горизонталі.

**Rest Timer Banner** — закріплений знизу над таб-баром: великий Bebas-відлік + кнопки −15с / +15с / Skip.

**Callout / Tip** — фон лайм 3%, ліва рамка 3px `lime`, Mono-заголовок + Caption-текст.

**Empty / Rest State** — велика напівпрозора емодзі/іконка + H2 (`faint`) + Caption.

## 5. Карта екранів (навігація)
Нижні вкладки:
1. **Сьогодні** (старт тренування) 2. **Історія** 3. **Аналітика** 4. **Тіло** 5. **Ще / Налаштування**
Бібліотека вправ і Деталі вправи відкриваються поверх (stack).

## 6. Вайрфрейми ключових екранів

**A. Сьогодні (Home)**
```

H1: СЬОГОДНІ
[ Stat: 4 дні ] [ 75 хв ] [ PPL+ ]
┌ Картка рутини дня ───────────┐
│ PUSH — Груди·Плечі·Трицепс   │
│ 6 вправ · ~75 хв             │
│        [ ПОЧАТИ ▸ ]          │  ← Primary
└──────────────────────────────┘
Останнє тренування: ВТ PULL  ›
```

**B. Активне тренування (ядро)**

## 7. Поради для Figma
- Створи **Color Styles** і **Text Styles** із таблиць вище — далі все по них.
- Збери компоненти з п.4 як **Components** з варіантами (active/inactive, done/empty).
- Працюй на одному артборді-фреймі, дублюй для кожного екрана.
- Шрифти: у Figma вони є в Google Fonts; у коді постав `@expo-google-fonts/bebas-neue`, `@expo-google-fonts/ibm-plex-sans`, `@expo-google-fonts/ibm-plex-mono` — вигляд збіжиться 1:1.
- Не вилизуй до пікселя: цей стиль навмисно «сирий/брутальний», головне — контраст, рамки, великі цифри.
